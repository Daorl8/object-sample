# object-sample — VESSEL (가구·오브제 편집숍)

이피웹(YIPI WEB) 포트폴리오용 ⑤-2번 샘플. 생활 오브제·가구 편집숍.

- **매트릭스 셀:** 제품그리드-퍼스트 · **매거진 그리드(비대칭 셀 스팬)** · 상단 수평 내비 · **라이트박스 퀵뷰**(QUINZE 호버리빌과 다른 시그니처) · 톤다운 에디토리얼
- **차별점(셀렉트 3종 중 2번):** QUINZE의 게이트·숫자인덱스·브루탈리즘을 의도적으로 회피 → 게이트 없음, 매거진 그리드, 라이트박스, **그레이지+잉크+클레이 액센트**의 차분한 무드, **에디토리얼 세리프(Newsreader)**.
- **정직 라벨:** 가짜 결제 없음 → 제품은 외부 스토어 연결, 도매·공간 컨설팅만 "문의" 폼. 모바일 하단바=PC CTA와 통일.
- **앵커 보정:** `scroll-padding-top` 적용.

배포: Cloudflare Workers. Formspree `YOUR_FORM_ID` 치환 필요.
